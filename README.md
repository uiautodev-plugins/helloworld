# helloworld

这是uiautodev的最小的插件示例。有一个文件是必须的: index.html

其中插件作为iframe网页嵌入到前端。

package.json包含插件的信息，里面所有的参数都是可选的。

完整版的package.json如下

```json
{
  "name": "helloworld",
  "displayName": "Hello World",
  "version": "0.0.1",
  "description": "Plugin of helloworld",
  "homepage": "https://github.com/uiautodev-plugins/helloworld"
}
```
